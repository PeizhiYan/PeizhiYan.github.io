Flipbook controls

  • Click the left / right half of the book to turn pages
  • ← → arrow keys to navigate  |  Home / End to jump to first / last
  • C = contents panel (page thumbnails)  |  T = cycle theme  |  A = toggle flip  |  F = fullscreen
  • Click and drag on a page to magnify — release to hide the magnifier
  • On mobile/tablet: swipe left/right to turn pages, hold+drag to magnify
  • The HTML file is fully self-contained — just open it in any browser
