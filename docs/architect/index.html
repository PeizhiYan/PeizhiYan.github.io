ArchitectHead supports continuous LOD control ranging from 0 (highest) to 1.0 (lowest). This figure shows the rendered image quality under different LOD settings. For each case, we provide a zoom-in view of selected regions. We recommend viewing the figure on a digital device with zoom for better inspection. The grey dots indicate the positions of the 3D Gaussian points. The test identity is taken from the NeRFace dataset. Red arrow indicates visible artifacts (sparse Gaussians) in the lowest LOD.
